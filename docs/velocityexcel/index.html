Installation

VelocityExcel is available as a NuGet package. Install version 1.0.2:

.NET CLI

      dotnet add package VelocityExcel --version 1.0.2

Nuget Package Manager Console

      Install-Package VelocityExcel -Version 1.0.2

PackageReference (csproj) 

      <PackageReference Include="VelocityExcel" Version="1.0.2" />
